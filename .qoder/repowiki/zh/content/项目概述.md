## 简介

Artisan 是一个企业级微前端基础平台脚手架，采用 Monorepo 架构，基于 qiankun 的 loadMicroApp 模式实现主应用对多类型子应用的统一加载与管理。该项目提供了完整的微前端解决方案，支持 Vue3/Vue2/iframe/link 四种应用类型，具备现代化技术栈和完善的工程化能力。

该项目的核心目标是为企业提供开箱即用的微前端基础设施，通过统一的管理平台实现多应用协同开发和部署，降低微前端架构的实施成本和技术门槛。

3. **监听器移除**: Iframe 应用卸载时移除 message 监听器

## 故障排查指南

### 常见问题及解决方案

#### 端口冲突问题

**问题症状**: 启动应用时报端口被占用错误

**解决方案**:
1. 检查端口占用情况：`lsof -i :端口号`
2. 修改应用配置文件中的端口号
3. 关闭占用端口的进程

#### CORS 跨域问题

**问题症状**: 控制台出现跨域错误

**解决方案**:
1. 确保子应用配置了正确的 CORS 响应头
2. 开发环境设置 `host: '0.0.0.0'` 和 `cors: true`
3. 配置正确的 `Access-Control-Allow-Origin` 值

#### 样式冲突问题

**问题症状**: 主应用样式被子应用覆盖

**解决方案**:
1. 使用 CSS Modules 或 Scoped CSS
2. 避免使用全局样式重置
3. 使用命名空间隔离样式

#### 路由跳转失败

**问题症状**: 点击链接后 URL 变化但页面不刷新

**解决方案**:
1. 确保 qiankun 环境使用 Memory/Abstract 路由模式
2. 独立运行使用 History 路由模式
3. 设置正确的 base 路径

## 结论

Artisan 微前端平台是一个功能完善、架构清晰的企业级解决方案。项目通过 Monorepo 架构实现了代码复用和统一管理，通过 qiankun 微前端框架提供了强大的应用集成能力。

### 主要优势

1. **开箱即用**: 提供完整的微前端解决方案，无需从零搭建
2. **多类型支持**: 支持 Vue3/Vue2/iframe/link 四种应用类型
3. **现代化技术栈**: 基于最新的前端技术，开发体验优秀
4. **完善的工程化**: 包含 CLI 工具、文档体系、质量保证
5. **企业级特性**: 跨应用通信、安全治理、性能优化等

### 适用场景

- 企业级后台管理系统
- 多产品线协同开发
- 第三方系统集成
- 微服务前端架构
- 大型复杂应用拆分

该项目为微前端架构的实施提供了良好的基础设施，降低了技术门槛，提高了开发效率，是企业数字化转型的理想选择。
